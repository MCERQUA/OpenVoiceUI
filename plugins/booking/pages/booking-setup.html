Cal.diy connection

1 Connect your Cal.diy account

Enter the API key and username for your account on the shared booking service (cal.jam-bot.com). Your key is stored on the server — never in the browser. Generate the key in Cal.diy under Settings → Developer → API keys.

The slug in your booking link — cal.jam-bot.com/<username>/30min
Used to verify incoming booking webhooks. Set this to the same secret you register in Cal.diy when you point a webhook at the URL below.
